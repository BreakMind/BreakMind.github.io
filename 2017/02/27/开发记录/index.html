2017-02-27 13:07:13.987 GooglyPuff[70647:1020285] App Transport Security has blocked a cleartext HTTP (http://) resource load since it is insecure. Temporary exceptions can be configured via your app’s Info.plist file.

说明：
从IOS9开始，强制使用了https协议，对于想要使用http协议，需要在info文件中添加如下字段：

在面板上点击右键，显示“Show Raws Keys/Vlaues”，然后增加下面内容
NSAppTransportSecurity
NSAllowsArbitraryLoads 设置为YES。

也可以在未选择“Show Raws Keys/Values”时，增加“App Transport Security Settings”，再增加子项“Allow Arbitrary Loads”，选择YES。

2017-02-27 13:07:22.203304 GooglyPuff[70647:1020281] [access] This app has crashed because it attempted to access privacy-sensitive data without a usage description. The app’s Info.plist must contain an NSPhotoLibraryUsageDescription key with a string value explaining to the user how the app uses this data.

说明：
这是禁止直接使用相册，需要在info.plist内增加NSPhotoLibraryUsageDescription，并添加说明。具体参照【修改1】
